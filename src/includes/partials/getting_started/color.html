Colors

All these colors are available in this platform

Usage
  • Primary color 1(#8dc63f) used for highlighting purpose and for the primary action buttons.
  • Unique color for parent companies.
  • Primary color 2(#63a042) used for the primary action button hover effect.
  • Each and every single text on the body should use the same color(#555555). Even the links.
  • A secondary button is when there is another action button beside the primary button and it will act the opposite(e.g. cancel). For the secondary action button use the secondery color 1 (#617d8c).
  • Confirmation messages like success, warning, attention and error background color should follow the chart above.
  • Platform body background color is light grey (#f1f2f7).
  • Cards will have solid white background color (#ffffff).
